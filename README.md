# Hand Gesture Recognition

This project implements a hand gesture recognition model using the LeapGestRecog dataset.

## 📌 Objective
To classify different hand gestures from images using machine learning.

## ⚙️ Tools Used
- Python
- OpenCV
- NumPy
- Scikit-learn

## 🧠 Model Used
Random Forest Classifier

## 📊 Dataset
LeapGestRecog dataset from Kaggle

## 📈 Output
The model achieved approximately 98% accuracy in classifying hand gestures.
